1. Настройка клиентской части (React)
  1.1. Установка Node.js и npm
  Убедитесь, что на вашем компьютере установлен Node.js и npm. Для проверки выполните команды:
  node -v
  npm -v
  Если Node.js не установлен, скачайте и установите его с официального сайта Node.js.

  1.2. Установка зависимостей
  Перейдите в корневую директорию проекта и установите все зависимости для фронтенда:
  cd frontend
  npm install
  Эта команда установит все библиотеки, указанные в файле package.json (например, React, необходимые хуки и стили).

  1.3. Запуск приложения
  Для запуска клиентской части в режиме разработки выполните команду:
  cd frontend
  npm start
  Это запустит приложение в режиме разработки. По умолчанию оно будет доступно по адресу http://localhost:3000.

2: Настройка серверной части
  2.1. Установка Node.js (если не установлено)
  Серверная часть также использует Node.js. Убедитесь, что он установлен, как описано в Шаге 1.1.

  2.2. Установка зависимостей для сервера
  Перейдите в корневую директорию серверного приложения и установите зависимости:
  cd backend
  npm install

  2.3. Запуск сервера
  Для запуска серверной части выполните:
  cd backend
  npm start
  Это запустит сервер на порту 8000, если он настроен по умолчанию.

3: Проверка работы приложения
  3.1. Проверка серверной части
  Убедитесь, что сервер работает на http://localhost:8000. Для этого можно использовать браузер, в строку запроса ввести:
  http://localhost:8000
  Ожидаемый результат — данные, возвращаемые сервером, которые клиент может использовать.

  3.2. Проверка клиентской части
  Перейдите в браузер по адресу http://localhost:3000 для проверки клиентской части. Приложение должно загрузиться, и вы сможете взаимодействовать с ним.

  3.3. Тестирование запросов
  В клиентской части в коде компонента App используются клавиши для взаимодействия с сервером. Например:
  Нажатие Enter загружает данные с сервера.
  Нажатие F7 отправляет данные на сервер. Убедитесь, что эти действия выполняются корректно, а серверная часть правильно обрабатывает запросы:
  На сервере в папке files в файле processedAds.json должен появиться массив с объектами - результатом обработки объявлений.

  P.S. 
  Сервер отдает объявления пакетами по 10 штук, после обработки первого пакета и отправки его на сервер, придет следующий пакет и т.д. пока не кончатся новые объявления.
  Если объявления кончатся, приложение выведет на экран сообщение: "Новых объявлений пока нет :)".
  Чтобы перезапустить приложение и получать объявления по новой необходимо выполнить команды на серверной части:
  ^C
  npm start
